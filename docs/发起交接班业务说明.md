# 接班医生
交班医生默认为当天值班人员，可修改，修改范围为当前科室人员。
# 患者人数情况总览
数据来源：患者总数：当前科室在院的患者总数。入院人数：入院时间24小以内，判断条件为visit.admission_datetime入院时间。转出人数：24小时以内转科表中，转出科室是当前科室的人数，判断依据转出科室transfer_record.from_dept_id。出院人数：出院时间为24小时以内的，就诊科室是当前科室的人数，判断条件为出院时间visit.discharge_datetime,就诊科室visit.dept_id。转入：转入时间为24小时以内的，就诊科室是当前科室的人数,判断条件转科记录表transfer_record.transfer_time转科时间，转入科室visit.to_dept_id等于当前科室。不显示死亡、手术、病危人数。
# 交班患者列表
交班患者：只显示在院的，入院时间24小时以内的患者和I级护理的患者。查询就诊信息表visit.admission_time入院时间，护理等级visit.nurse_level_code.
“目前情况”，只显示当前患者24小时以内的临时医嘱显示出来，从医嘱表order_main,临时标识order_main.order_type='N',医嘱时间判断order_main.start_time
