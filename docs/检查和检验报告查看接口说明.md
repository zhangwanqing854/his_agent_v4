# 检查和检验报告查看
- 发起报告-》患者列表-〉操作列，增加查看检查和检验报告按钮。点击后新打开页面
- 检查报告接口：
地址：http://sichuangpacs.pkuih.edu.cn/IntegrationCenter/index.html#/views/eipTimeLine2?CMD=showlist&PW=webweb&DOMAINCODE=patientid&DOMAINID={{:patient_no}}
说明：{{:patient_no}}是变量，取该患者的患者编码patient_no
- 检验报告接口：
http://10.2.48.64:8001/cdr/login/loginiihPortalIntegrated.html?viewId=V002&patientId={{:patient_no}}&domainId=02&styleId=01&display=0&userId={{:userId}}&visitTimes=&XExternalUrlFlag=1&systemId=SIIH&bsiihtype=0&download=1&visitTimes=
说明：{{:patient_no}}是变量，取该患者的患者编码patient_no。
{{:userId}}是变量，取当前登录医生的用户编码usercode
# 检查和检验报告查看的地址配置
地址需要支持配置，在系统配置界面新增加检查报告和检验报告查看的地址配置。
