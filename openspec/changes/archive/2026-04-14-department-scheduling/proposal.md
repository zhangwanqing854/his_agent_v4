## 为什么

当前系统缺少科室排班管理功能，护士长无法便捷地安排科室人员的值班计划。每次交接班时需要手动确认值班人员，效率低下且容易出错。引入排班功能可提升交接班流程的规范性和可追溯性。

## 变更内容

新增科室排班管理模块，支持：
1. **月度排班管理** - 按月创建和维护排班计划
2. **排班人员选择** - 从当前科室工作人员中选择值班人员
3. **卡片视图展示** - 以卡片形式直观展示排班信息
4. **打印功能** - 支持卡片式打印，一张 A4 纸打印两个月排班

## 功能 (Capabilities)

### 新增功能

- `department-scheduling`: 科室排班管理功能，包含月度排班计划创建、人员安排、卡片视图展示和打印输出

### 修改功能

无 - 此变更不涉及现有规范的行为变更。

## 影响

| 影响范围 | 说明 |
|---------|------|
| 前端页面 | 新增排班管理页面（SchedulingManagement.vue）、排班卡片组件、打印模板 |
| 后端 API | 新增排班相关 Controller、Service、Mapper |
| 数据库 | 新增排班表（scheduling）、排班明细表（scheduling_detail） |
| 路由配置 | 新增 /scheduling 路由 |

**与 HIS/LIS/PACS 系统对接点**: 无直接对接。排班人员信息来源于系统内部的科室人员管理（HisStaffManagement），不涉及外部 HIS 系统排班数据同步。

## 非目标

- 不实现排班自动推荐或智能排班算法
- 不支持跨科室排班（仅限当前登录科室）
- 不实现排班与 HIS 系统的双向同步
- 不支持排班模板导入导出（本期不做）
