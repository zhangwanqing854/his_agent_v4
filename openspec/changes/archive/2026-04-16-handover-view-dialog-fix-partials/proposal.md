## 为什么

交班管理列表页面的"查看"按钮点击无反应，用户无法查看交班详情。此问题阻止了核心功能的正常使用，需要立即修复以恢复交班查看功能。

## 变更内容

1. 修复 HandoverManagement.vue 中"查看"按钮的事件处理问题
2. 优化交班详情弹窗的显示逻辑
3. 改进错误提示和加载状态显示

## 功能 (Capabilities)

### 新增功能

- `handover-view-dialog`: 交班详情查看弹窗功能，支持显示交班摘要、患者列表，并可打印交班报告

### 修改功能

无（修复现有功能的行为问题，不涉及需求变更）

## 影响

- **前端代码**: `frontend/src/views/HandoverManagement.vue` - 修复 handleView 函数和按钮事件处理
- **UI交互**: 交班详情弹窗正常显示，用户可查看交班患者列表
- **非目标**: 不涉及后端 API 变更、不涉及数据库变更、不涉及新增页面路由
