## MODIFIED Requirements

### 需求:语音录入必须使用百度离线语音 SDK

语音录入功能必须从 Web Speech API 切换为百度离线语音 SDK，以支持离线环境使用。

**原有实现：** 使用浏览器原生 Web Speech API（window.SpeechRecognition 或 window.webkitSpeechRecognition）

**新实现：** 使用百度离线语音 Web SDK

#### 场景:单人语音录入使用百度 SDK
- **当** 用户点击患者行的麦克风按钮进行单人语音录入
- **那么** 系统调用百度离线语音 SDK 进行识别，返回识别文本写入需观察项字段

#### 场景:批量语音录入使用百度 SDK
- **当** 用户点击批量语音录入按钮
- **那么** 系统调用百度离线语音 SDK 进行连续识别，返回识别文本进行批量匹配

#### 场景:语音识别失败降级处理
- **当** 百度离线语音 SDK 初始化失败或识别出错
- **那么** 系统显示错误提示，建议用户检查模型文件或手动输入
