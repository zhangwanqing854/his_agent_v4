## 修改需求

### 需求:语音录入必须使用WebSocket连接

语音录入功能必须使用 WebSocket 连接到后端 Vosk 语音识别服务，不再使用浏览器原生 Web Speech API。

#### 场景:WebSocket语音识别启动
- **当** 用户点击开始录音按钮
- **那么** 系统建立 WebSocket 连接，开始录制并发送音频数据

#### 场景:WebSocket语音识别实时显示
- **当** 后端返回临时识别结果（partial）
- **那么** 前端实时显示临时识别文本

#### 场景:WebSocket语音识别完成
- **当** 用户点击完成按钮或发送结束信号
- **那么** 后端返回最终识别结果，前端显示完整文本
